# 💄 Cosmetician Business Website (Freelance Project)

A professional, high-end landing page and business website developed for a private cosmetician. This project was built as a freelance engagement, focusing on elegant UI/UX, service showcasing, and mobile responsiveness.

---

## 🚀 Project Overview

The goal was to create a digital presence for a beauty business that reflects professionalism and aesthetic appeal. The site provides customers with information about treatments, pricing, and an easy way to get in touch.

## ✨ Key Features

* **Premium UI/UX:** Clean and elegant design tailored for the beauty industry using Tailwind CSS.
* **Fully Responsive:** Optimized for a seamless experience on smartphones, tablets, and desktops.
* **Service Gallery:** A dynamic display of treatments and services offered.
* **Contact Integration:** Quick-access links for booking appointments via WhatsApp/Phone.
* **Performance Optimized:** Fast loading times using React's efficient rendering.

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS (Custom color palette and typography)
* **Icons:** Lucide-React / FontAwesome
* **Deployment:** Vercel / Netlify
